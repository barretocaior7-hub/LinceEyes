# 🛡️ HealthTech - Sistema Biométrico de Controle de Acesso Web

Aplicação para controle de acesso físico/lógico em laboratórios de pesquisas clínicas utilizando Inteligência Artificial executada no navegador (*Client-Side AI*) com `face-api.js` e backend em `Node.js`.
